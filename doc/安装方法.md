# 安装方法

## 项目配置
建议采用goland IDE 
ADD Configuration->左上角+->go build类型
右侧填写
files :{你的项目路径}/hellox.x/main.go
workdir:{你的项目路径}/hellox.x/

将asset目录复制到对应项目下
如将asset复制到hellox.x下
